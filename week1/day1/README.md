pwd 
in ra link thư mục mình đang trỏ tới 

cd .. 
lui về thư mục mẹ 

ls -la 
liệt kê các file và thư mục đang trỏ tới 

tree -L 2 /etc 
đưa ra thông tin của thư mục con ở cấp con 

uname -a 
thông tin của kernel 

touch file.txt 
tạo file rỗng có tên là file.txt 

mkdir 
tạo thư mục nếu muốn lòng nhiều thư mục thì mkdir -p a/b/c 

cp file.txt /tmp/ 
copy file.txt và file đó sẽ được lưu vào bộ nhớ tạm thời tmp 

rm file.txt xóa file, nếu muốn xóa thư mục thì rm -rf folder/
